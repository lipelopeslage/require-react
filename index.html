Require+React

Exemplo prático utilizando React com Require.js.

Versões de módulos NPM utilizadas neste exemplo:

  • React-addons/React-DOM: ~0.14.7
  • RequireJS: 2.2.0

Este exemplo mostra como combinar módulos carregados via RequireJS e componentes React.

Clique aqui e veja o exemplo.

Os componentes ReactJS são escritos de acordo com o formato JSX. Veja um exemplo de componente abaixo:

var Component = React.createClass({
        render: function(){
            return (
                <div>
                    Este é um componente React =)
                </div>
            );
        }
    });
    ReactDOM.render(
        <Component/>, document.querySelector('main')
    );

Referências:

  • npm para dependências (package.json)
  • require.js para carregar os módulos
  • react para criação de componentes em JSX
  • jsx-requirejs-plugin torna possível a integração de arquivos JSX e o framework requirejs
